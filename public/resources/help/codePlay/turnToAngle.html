שלב 8 - שינוי כיוון העיפרון לזוית מסויימת

מטרת השלב

ציור קווים בכיוונים נוספים ע"י שינוי כיוון העפרון בזוית מסויימת.

הוראות

קבוצת הוראות התנועה מכילה כעת גם הוראה לשינוי כיוון העיפרון לזוית מסויימת בין 0 ו-360. 0 משמעותו ימינה, 90 למעלה, 180 שמאלה ו-360 ימינה (כמו 0). ניתן לקבוע את הזוויות לכל מספר בין 0 ו-360 למשל, 45 היא הזוית שבאמצע בין 0 ו-90, כלומר אלכסון למעלה וימינה. העיפרון יפנה לכוון המתאים לזוית שנבחרה. הוראה זאת מסובבת את העפרון אך לא גורמת לציור קו. על מנת לצייר יש להוסיף הוראת "התקדם". כשתבצע את ההוראות שמופיעות על המסך תראה שמצוייר קו בזוית 45 מעלות.
